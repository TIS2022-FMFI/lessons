# fem
Registry of FEM calculations

Application serves for searching lessons by categories and keywords

Category- basic CRUD operations, getters, setters
Key_word- basic CRUD operations, getters, setters
Key_word_problem- basic CRUD operations, getters, setters
Problem- basic CRUD operations, getters, setters

CategoryFinder- 
    findById(id)- find category from database by id
    findAll()- find all categories
KeywordFinder-  
    findById(id)- find keyword from database by id
    findByTitle(titleK)- find keyword by title 
    findByPrime(prime)- find prime/no-prime keywords
